# OverView
ExpressとMongoDBを利用したCRUDなアプリケーションを制作する。

## CRUD（クラッド）
ほぼ全てのコンピュータソフトウェアが持つ、以下の4つの基本機能のイニシャルを並べた用語。

- Create（生成）
- Read（読み取り）
- Update（更新）
- Delete（削除）

CRUDのそれぞれの機能を網羅していることは、データベースシステムの完全性を備えるために必須であるとされている。
